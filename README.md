# Personal Site

Personal blog and portfolio — [ajkamel.github.io](https://ajkamel.github.io)

## Quickstart

**Prerequisites:** Ruby and Bundler installed.

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve
```

Site will be available at `http://127.0.0.1:4000/`.
